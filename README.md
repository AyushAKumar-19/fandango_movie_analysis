# Fandango Movie Ratings Analysis

In October 2015, [FiveThirtyEight](https://fivethirtyeight.com) published a [story](https://fivethirtyeight.com/features/fandango-movies-ratings/) calling Fandango's metrics on user ratings to question.
This project analyses and tries to verify this story with the help of data (Source: [Github](https://github.com/fivethirtyeight/data/tree/master/fandango)) using python and data science libraries like Numpy, Pandas, Matplotlib and Seaborn.
